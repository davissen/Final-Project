Welcome to the Space Invaderz game

Enjoy the flight

Hello my name is Davissen Moorgan, I am a student at the NYCDA. I have created a one level version of the now cult Space Invaders video Game. For this project I have explore Phaser and followed the tutorials in order to understand the various options and functionalities that Phaser offers.

In order to build the game a local server environment software was required by the phaser. For windows users, WAMP Server and XAMPP were recommended. For OS X users, MAMP was recommeded.
